WebWeekly
=========

网站开发周刊

收集整理互联网跟网站开发相关的资源，每周1次以邮件的方式发送。对往期发布的如果是专题的后期还会补充和完善，最新的更新都会在`github`上。

## 为何做这个

平时更多的是`input`，希望可以通过这个平台将可以有点自己的`ouput`。分享自己认为不错的资源给大家。内容更加偏向于专题。
